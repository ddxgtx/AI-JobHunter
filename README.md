# 🎯 AI-JobHunter

**智能求职助手 Chrome 扩展**

一款智能 Chrome 浏览器扩展，帮助求职者高效求职。支持 Boss 直聘、前程无忧、智联招聘等 7 大招聘平台，基于职位描述（JD）自动生成个性化话术，并提供 AI 智能填写简历、面试准备、关键词优化等一站式求职辅助功能。

## ✨ 功能特性

### 🚀 核心功能

| 功能 | 描述 |
|------|------|
| **智能话术生成** | 根据 JD 和简历自动生成打招呼话术，支持 5 种风格（专业/亲和/自信/简洁/热情）和 3 种长度（简短/适中/详细） |
| **JD 智能提取** | 一键从当前招聘页面提取职位描述，内置反爬虫噪声清除和内容清洗算法 |
| **AI 清洗 JD** | 可选开关，调用 API 对提取的 JD 进行二次清洗，去除无关信息 |
| **职位信息卡片** | 自动提取岗位名称、公司、薪资、地点、经验/学历要求并以标签展示 |

### 📄 简历管理

| 功能 | 描述 |
|------|------|
| **简历信息表单** | 13 个字段，支持保存/导出/导入 JSON 格式 |
| **PDF 简历导入** | 支持上传 PDF 简历，AI 自动解析填充表单 |
| **Markdown 简历** | 一键生成 Markdown 格式简历文档，支持编辑/复制/保存 |
| **AI 智能填写** | 扫描招聘页面表单字段，AI 自动匹配简历信息并填写 |

### 🤖 AI 分析工具

| 工具 | 功能 |
|------|------|
| **关键技能提取** | 从 JD 中提取关键技能要求，以标签形式展示 |
| **简历匹配分析** | AI 评估简历与 JD 的匹配度，显示匹配/缺少技能和优化建议 |
| **面试准备** | 根据 JD 生成 8-12 个分类面试问题（技术基础/项目经验/系统设计/行为面试），附带回答提示 |
| **关键词优化** | 分析简历与 JD 的关键词匹配情况，提供优化建议和改进后的个人优势摘要 |
| **求职信生成** | 基于 JD 和简历生成定制求职信，支持风格选择 |
| **薪资分析** | AI 推断薪资范围，提供市场趋势和谈判建议 |

### 💡 其他特性

- **历史记录** - 保存最近 20 条话术生成记录，支持导出为 TXT 文件
- **多模型支持** - 支持 OpenAI、DeepSeek 及自定义 API 端点
- **流式输出** - 话术生成采用流式传输，实时显示结果
- **侧边栏模式** - 以 Chrome Side Panel 形式运行，不影响页面浏览
- **浮动按钮** - 支持拖拽的浮动入口按钮，点击打开侧边栏

## 📦 安装

### 方式一：开发者模式安装（推荐）

```bash
# 1. 克隆仓库
git clone https://github.com/ddxgtx/AI-JobHunter.git

# 2. 打开 Chrome 扩展管理页面
# 地址栏输入: chrome://extensions/

# 3. 开启右上角「开发者模式」

# 4. 点击「加载已解压的扩展程序」，选择项目目录
```

### 方式二：下载 ZIP

1. 点击页面右上角 **Code** → **Download ZIP**
2. 解压下载的文件
3. 按照方式一的步骤 2-4 操作

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + G` | 打开侧边栏 |

## 🚀 使用方法

### 基本流程

```
┌─────────────────────────────────────────────────────────────┐
│  1. 配置 API Key                                            │
│     设置 → API 配置 → 选择模型 → 输入 Key → 保存            │
├─────────────────────────────────────────────────────────────┤
│  2. 填写简历信息                                            │
│     切换到「简历填写」标签页，填写个人信息                    │
├─────────────────────────────────────────────────────────────┤
│  3. 获取职位描述                                            │
│     打开招聘平台职位详情页 → 点击「从当前页面获取 JD」       │
├─────────────────────────────────────────────────────────────┤
│  4. 生成话术                                                │
│     选择风格和长度 → 点击「生成打招呼话术」                  │
└─────────────────────────────────────────────────────────────┘
```

### AI 工具使用

| 工具 | 使用方法 |
|------|----------|
| **提取关键技能** | 获取 JD 后点击「提取关键技能」按钮 |
| **匹配分析** | 需先保存简历，点击「匹配分析」查看匹配度 |
| **面试准备** | 点击「面试准备」生成面试问题 |
| **关键词优化** | 点击「关键词优化」获取简历优化建议 |
| **求职信** | 点击「求职信」生成定制求职信 |
| **薪资分析** | 点击「薪资分析」查看薪资范围和建议 |

### 智能填写

1. 打开招聘平台的简历填写页面
2. 切换到「简历填写」标签页
3. 确保简历信息已保存
4. 点击「AI 智能填写当前页面」

## ❓ 常见问题

### Q: 如何获取 API Key？

**OpenAI:**
1. 访问 https://platform.openai.com/
2. 注册/登录后，在 API Keys 页面创建新密钥

**DeepSeek:**
1. 访问 https://platform.deepseek.com/
2. 注册/登录后，在 API Keys 页面创建新密钥

### Q: 支持哪些 AI 模型？

- OpenAI: GPT-3.5-turbo, GPT-4, GPT-4-turbo
- DeepSeek: DeepSeek-Chat
- 自定义: 任何兼容 OpenAI API 格式的端点

### Q: 话术生成没有反应？

1. 检查 API Key 是否正确配置
2. 确认网络连接正常
3. 查看浏览器控制台是否有错误信息

### Q: JD 提取失败？

1. 确认当前页面是支持的招聘平台
2. 等待页面完全加载后再提取
3. 尝试刷新页面后重试
