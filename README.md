# AI-JobHunter

一款智能 Chrome 浏览器扩展，帮助求职者高效求职。支持 Boss 直聘、前程无忧、智联招聘等 7 大招聘平台，基于职位描述（JD）自动生成个性化话术，并提供 AI 智能填写简历、面试准备、关键词优化、职位评估、求职管道追踪等一站式求职辅助功能。

> 灵感来自 [Career-Ops](https://github.com/santifer/career-ops) —— 将 AI 求职指挥中心带到浏览器端。

## 功能特性

### 核心功能

- **智能话术生成** - 根据 JD 和简历自动生成打招呼话术，支持 5 种风格（专业/亲和/自信/简洁/热情）和 3 种长度（简短/适中/详细）
- **JD 智能提取** - 一键从当前招聘页面提取职位描述，内置反爬虫噪声清除和内容清洗算法
- **AI 清洗 JD** - 可选开关，调用 API 对提取的 JD 进行二次清洗
- **职位信息卡片** - 自动提取岗位名称、公司、薪资、地点、经验/学历要求

### AI 分析工具

- **关键技能提取** - 从 JD 中提取关键技能要求
- **简历匹配分析** - AI 评估简历与 JD 的匹配度，显示匹配/缺少技能和优化建议
- **职位评估评分** - 多维度 A-F 评分（技术匹配/经验匹配/薪资匹配/学历匹配/地域匹配/成长空间/公司前景），加权综合评分
- **面试准备** - 根据 JD 生成 8-12 个分类面试问题（技术基础/项目经验/系统设计/行为面试），附带回答提示
- **关键词优化** - 分析简历与 JD 的关键词匹配，提供优化建议和改进后的个人优势摘要
- **求职信生成** - 基于 JD 和简历生成定制求职信
- **薪资分析** - AI 推断薪资范围，提供市场趋势和谈判建议
- **深度公司研究** - 从 JD 推断公司信息：行业、技术栈、文化特点、优势/风险、面试建议
- **LinkedIn 外联** - 生成 LinkedIn 联系人消息，支持风格选择

### 简历管理

- **简历信息表单** - 13 个字段，支持保存/导出/导入 JSON
- **PDF 简历导入** - 上传 PDF 简历，AI 自动解析填充
- **Markdown 简历** - 一键生成 MD 格式简历文档
- **AI 智能填写** - 扫描页面表单字段，AI 自动匹配并填写

### 求职管道

- **管道追踪** - 追踪求职状态：感兴趣 → 已投递 → 面试中 → 已录取 → 已拒绝
- **状态筛选** - 按状态筛选查看求职记录
- **导出记录** - 导出求职管道为 TXT 文件

### 其他

- **历史记录** - 保存最近 20 条话术生成记录，支持导出
- **多模型支持** - OpenAI / DeepSeek / 自定义 API 端点
- **流式输出** - 话术生成实时显示
- **侧边栏模式** - Chrome Side Panel 运行
- **浮动按钮** - 可拖拽的浮动入口

## 安装

1. 下载或克隆本仓库
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择项目目录
5. 点击扩展图标或按 `Alt+G` 打开侧边栏

## 使用方法

### 基本流程

1. 配置 API Key（设置 → API 配置）
2. 填写简历信息（简历填写标签页）
3. 打开招聘平台职位详情页
4. 点击「从当前页面获取 JD」
5. 选择话术风格和长度，生成话术

### AI 工具

获取 JD 后，可用的 AI 工具按钮：

| 按钮 | 功能 |
|------|------|
| 🏷️ 提取关键技能 | 从 JD 提取技能关键词 |
| 📊 匹配分析 | 简历与 JD 的匹配度评估 |
| 🎤 面试准备 | 生成面试问题和回答提示 |
| 🔑 关键词优化 | 简历关键词优化建议 |
| ✉️ 求职信 | 生成定制求职信 |
| 💰 薪资分析 | 薪资范围和谈判建议 |
| 💼 LinkedIn | 生成 LinkedIn 外联消息 |
| 🏢 公司研究 | 公司信息和面试建议 |
| 📋 职位评估 | 多维度 A-F 评分 |

### 求职管道

切换到「求职」标签页：
- 点击「+ 记录当前职位」添加到管道
- 点击记录可切换状态（感兴趣 → 已投递 → 面试中 → 已录取 → 已拒绝）
- 使用状态筛选查看不同阶段的记录

## 技术栈

- Chrome Manifest V3
- Chrome Side Panel API
- Chrome Scripting API
- OpenAI / DeepSeek / 自定义 API
